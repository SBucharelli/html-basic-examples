Already Making Progress

The following is a list of elements/attributes that I've incorporated into this html file:

dl
The dl tag allows me to create a definition or description list. It is used to group all
the terms within the list together.
dt
The dt tag allows for a definition term to be created. Every term that is going to be
defined/elaborated on will be enclosed within dt tags.
dd
The dd tag is used to define a term within the list. Each definition within this list will
be wrapped in between dd element tags.
code
The code tag is how I am displaying the element names to look like "code" without
the computer treating them as such. Every term in this list will be enclosed in code
element tags.
h1
The h1 tag is typically used to display the heading for a page. It is the largest section
heading of the 6 available sizes (h1 through h6) and it generally defines the most
important heading. I put the title of this page in h1 tags.
h3
The h3 tag is used to display a heading that is two levels smaller than an h1. I wrapped
my sub-section headings in the h3 element tags. I like pairing the h1 and h3 tags due
to the size difference.
p
The p tag is used to define paragraphs of text. This is nice to create spacing between
the text of the body. I utilized the p tag in 'The Beginning' section above to space out
my introductory message.
a
The a tag, or anchor tag, is what I used to create a link to my About Me markdown file
at the beginning of this portfolio page.
href
The href tag is an attribute of the anchor tag. I used the href to specifically set up the
path to access my About Me file.
title
The title tag was utilized twice in this file. The first instance was a form of metadata.
It was used in the head section of the html file in order to title the document itself. This
shows up in the tab of the webpage in the browser. The second instance of when I used
the title tag was as an attribute of the achor tag. When used as an attribute for the
anchor tag, it provides a title for the hyperlink if a mouse is hovering over it.
div
The div tag is a block-level element that I used break up the page into to smaller
contained sections. Each section that I wanted seperated, I wrapped inside div tags.
br
The br tag is an element that causes a line break within text. I used the br tag
numerous times throughout this html file in order to push parts of sentences to the next
line. As a result, I was able to format the page into a more compact look. An intersting
bit about the br tag is that it is self-closing, therefore, you only need one tag to insert a
line break.
small
The small tag is used to define text that should be smaller. I gave this a try by adding a
little blurb at the end of the html file stating when I started working on this portfolio. I
wrapped it in between small tags so that the text was smaller and not obtrusive since it
was a quick origination comment.
